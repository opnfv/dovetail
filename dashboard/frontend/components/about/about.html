Overview

Dovetail tool intends on being THE source of tools to provide evaluation of the use of OPNFV trademarks.

Dovetail tool provides users with a wrapper which leverage other test projects, that helps to provide evaluation of the use of OPNFV trademarks. It does so by validating any NFV(Network Function Virtualization) implementation against the test cases in Dovetail.
